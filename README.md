# HTML5PieChart
2D Pie chart widget using HTML5 Canvas and JavaScript

Sample usage
```
var data = [
	{
		"name": "Item One",
		"value": 50
	},
	{
		"name": "Item Two",
		"value": 20
	},
	{
		"name": "Item Three",
		"value": 30
	}
];

new PieChart(data);

// or

var myVar = new PieChart(data);

```

See test.html for example
